# SharedElementImageCircleToSquare
点击一个activity中的圆形头像imageview，跳转到第二个页面，第二个页面是大的方形头像。
采用安卓提供的SharedElement来实现无缝过渡，圆形渐变成方形。
网上找个半天没有发现有类似的实现，于是自己动手实现了一个
